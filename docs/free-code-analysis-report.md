## 1. 项目概述

### 1.1 项目定位

**free-code** 是一个类 Claude Code 的迷你 Agent 学习项目，基于 TypeScript/Bun 构建的 CLI 工具，提供交互式 AI 编程助手功能。

### 1.2 主要功能

| 功能 | 描述 |
|------|------|
| **交互式 REPL** | 基于 Ink/React 的终端 UI，提供富交互体验 |
| **AI 对话** | 支持多轮对话，集成 Anthropic Claude API |
| **工具系统** | 可扩展的工具调用机制（文件操作、命令执行、Agent 等） |
| **命令系统** | 斜杠命令（/command）支持各种快捷操作 |
| **MCP 集成** | Model Context Protocol 支持，可连接外部服务 |
| **插件系统** | 支持加载和管理插件 |
| **技能系统** | 可复用的技能模块 |
| **Bridge 模式** | 远程控制和 IDE 集成 |
| **多模型支持** | 支持 Anthropic、OpenAI、AWS Bedrock、Google Vertex 等 |

## 7. 关键发现

### 7.1 架构亮点

1. **分层清晰**: 表现层 → 应用层 → 领域层 → 基础设施层
2. **生成器模式**: 使用 AsyncGenerator 实现流式消息处理
3. **依赖注入**: QueryEngine 通过配置对象注入依赖
4. **动态导入**: 优化启动性能，按需加载模块

### 7.2 扩展机制

1. **命令扩展**: 在 `src/commands/` 创建命令目录 → 在 `src/commands.ts` 注册
2. **工具扩展**: 在 `src/tools/` 创建工具目录 → 在 `src/tools.ts` 注册
3. **技能扩展**: 在 `src/skills/` 创建技能 → 在 `src/skills/bundled/index.ts` 注册

### 7.3 安全特性

- **API 密钥管理**: 本地存储，加密传输
- **OAuth 流程**: 标准 PKCE 流程
- **权限控制**: 细粒度工具权限 (low/medium/high)
- **沙箱执行**: Bash 命令隔离

### 7.4 性能优化

- **快速路径**: CLI 入口支持多种快速路径（version、daemon 等）
- **并行初始化**: main.tsx 中并行启动多个初始化任务
- **特征标志**: 使用 `feature()` 函数进行编译时死代码消除
